/architecture

Inspired by MetaGPT, ChatDev, and Claudish—rebuilt for total autonomy.

Multi-Model Synergy

Rotate, mix deploy-multi agent (if needed) basically use multiple Gemini, OpenAI, Claude, OpenRouter, etc. API keys alongside other provider models to bypass limits. Let the strongest model manage the complex logic while cheaper models handle the grunt work.

Tailscale + Ollama/Local

Already integrated. Wire your local, uncensored Ollama or any local LLM instances directly into the office swarm via Tailscale. Keep sensitive data processing strictly on-premise.

Human-in-the-Loop

They do the work, you hold the leash. A comprehensive monitoring dashboard allows you to oversee agent collaboration, correct course, and approve deployments.
